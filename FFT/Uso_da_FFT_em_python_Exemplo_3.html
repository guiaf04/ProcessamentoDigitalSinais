Transformada de Fourier Discreta de funes retangulares

Esse notebook tem por objetivo demonstrar e explicar como funcionada a Transformada de Fourier Discreta aplicada para funes retngulares, que so fundamentais no processamento digital de sinais.

Uma funo retangular pode ser definida como N amostras contem K valores unitrios em sequncia, com 1 K N, de forma que na forma de frequncia, temos:

$X(m) = \sum_{n=-(N/2)+1}^{N/2}x(n)e^{-j2\pi nm/N}$

Alm disso, a partir de manipulaes matemticas, temos que:

$X(m) = e^{j(2\pi m/N)(n_o - (K-1)/2)} \frac{sin(\pi mK/N)}{sin(\pi m/N)}$

onde $-n_0$ o menor valor onde o sinal 1.

A ttulo de comparao, vamos utilizar a transformada por meio da equao acima, chamaada equao de Dirichlet, com a transformada obtida por meio do numpy.

Dessa forma, considere um sinal com uma quantidade K = 11 de pulsos contnuos, comeando em $-n_0 = 0$ com N = 40 amostras, como demonstra a figura abaixo:

In[3]:
N = 40  # Nmero total de amostras
K = 11  # Nmero de amostras iguais a 1
n0 = 0  # Incio das amostras iguais a 1

n_inicio = int(-N/2 + 1)
n_fim = int(N/2)
n = np.arange(n_inicio, n_fim + 1)

# 2. Cria o sinal x(n) inicializado com zeros
x = np.zeros_like(n)

# 3. Define os limites onde o sinal  1
inicio_pulso = -n0
fim_pulso = -n0 + K - 1

# 4. Define o valor 1 para o intervalo do pulso retangular
# Usamos uma condio lgica para encontrar os ndices corretos em 'n'
condicao = (n >= inicio_pulso) & (n <= fim_pulso)
x[condicao] = 1

# --- Plotagem do Grfico ---

# Configura a figura e os eixos
plt.figure(figsize=(12, 6))

plt.title(f'Sinal Retangular Discreto $x(n)$', fontsize=16)

plt.ylabel('$x(n)$ (Amplitude)', fontsize=12)
plt.stem(n, x)
plt.yticks([0, 1]) # Mostra apenas os ticks 0 e 1 no eixo y

Em posse desse sinal, vamos comparar os sinais utilizando o numpy e a funo de Dirchelet, para entender melhor essa DFT

In[4]:
# Calcula a DFT usando numpy
X_numpy = np.fft.fft(x)

# Calcula a DFT usando a frmula de Dirichlet
X_dirichlet = np.zeros(len(n), dtype=complex)
for m in range(N):
    if m % (N / K) == 0 and K != N:
        # Caso especial para evitar diviso por zero
        X_dirichlet[m] = K
    elif m == (N / 2):
        #Caso especial para m = 0
        X_dirichlet[m] = K
    else:
        denominator = np.sin(pi * m / N)
        if denominator == 0:
            # Handle the case where the denominator is zero gracefully
            X_dirichlet[m] = K if m % (N / K) == 0 else 0
        else:
            X_dirichlet[m] = np.exp(1j * (2 * pi * m / N) * (n0 - (K - 1) / 2)) * (np.sin(pi * m * K / N) / denominator)

Como podemos ver abaixo, a magnitude da funo nos dois mtodos se mantm iguais, com exceo do ponto 0 que ser explorado na frente. No entanto, a fase de cada mtodo diferem, devido ao fato da numpy no refletir exatamente os lobos secundrios entre os lobos principais da DFT gerada.
